Basic Shortcuts

  • Ctrl + C: Copy selected text or item.
  • Ctrl + X: Cut selected text or item.
  • Ctrl + V: Paste copied or cut text or item.
  • Ctrl + Z: Undo an action.
  • Ctrl + Y: Redo an action.

Advanced Shortcuts

  • Win + L: Lock your PC.
  • Win + D: Display and hide the desktop.
  • Alt + Tab: Switch between open apps.
  • Ctrl + Shift + Esc: Open Task Manager.
  • Win + R: Open the Run dialog box.

File Explorer Shortcuts

  • Win + E: Open File Explorer.
  • Ctrl + N: Open a new window.
  • Ctrl + W: Close the current window.
  • Alt + D: Select the address bar.
  • Ctrl + Shift + N: Create a new folder.

Virtual Desktop Shortcuts

  • Win + Ctrl + D: Add a virtual desktop.
  • Win + Ctrl + F4: Close the current virtual desktop.
  • Win + Ctrl + Left/Right Arrow: Switch between virtual desktops.

Command Prompt Shortcuts

  • Ctrl + A: Select all text in the current line.
  • Ctrl + C: Copy selected text.
  • Ctrl + V: Paste copied text.
  • Up/Down Arrow: Navigate command history.
  • Ctrl + M: Enter Mark mode.
