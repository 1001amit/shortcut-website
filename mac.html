Basic Shortcuts

  • Command + C: Copy selected text or item.
  • Command + X: Cut selected text or item.
  • Command + V: Paste copied or cut text or item.
  • Command + Z: Undo an action.
  • Command + Shift + Z: Redo an action.

Advanced Shortcuts

  • Command + Space: Open Spotlight search.
  • Command + Tab: Switch between open apps.
  • Command + Option + Esc: Force quit an app.
  • Command + Shift + 3: Take a screenshot.
  • Command + Shift + 4: Take a screenshot of a selected area.

Finder Shortcuts

  • Command + N: Open a new Finder window.
  • Command + W: Close the current Finder window.
  • Command + Option + W: Close all Finder windows.
  • Command + Shift + N: Create a new folder.
  • Command + Delete: Move selected item to the Trash.

System Shortcuts

  • Command + Shift + Q: Log out of your macOS user account.
  • Command + Option + Eject: Put your Mac to sleep.
  • Command + Control + Power: Force your Mac to restart.
  • Command + Control + Q: Lock your screen.
  • Command + Option + Power: Put your Mac to sleep.
